# Atya.Foundation.Abstractions

Provider-neutral contracts and marker interfaces for Atya foundation packages.

## Included contracts

- `IHasId<TId>`
- `IEntity<TId>`
- `IAuditable`
- `ISoftDeletable`
- `IPagedQuery`
- `IResult`

## Installation

```bash
dotnet add package Atya.Foundation.Abstractions
```
